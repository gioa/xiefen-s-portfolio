Opera Sphinx Platform

Project Background:
The lack of smooth experience in a mobile HTML5 game is always a big problem for game developers. To make the performance of web game like that of a native game and save game developers from worrying about performance issue, we build Sphinx, a game engine particularly optimized for HTML5 games.


Role: Product Manager

Teams: Product, Operation, Engineering, Design, Marketing

Duration: 4 months on and off


Overview:

The users of Sphinx are mainly game developers, they focus on how Sphinx could help them and how easy to start with. They are also interested in the technical details behind the product. So accordingly we provided them a web platform, a mobile emulator for debugging, and a packaging platform.
