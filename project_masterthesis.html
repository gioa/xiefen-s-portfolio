Master Thesis: Context Aware Framework

Project Background:
Context aware means applications can both sense and react based on the environment. But in reality the commercial context aware mobile applications are still limited, mostly are those using location information to find nearby friends or services available in the market.


This happened mainly because of a lack of an effective mechanism to collect and share contextual information. Nowadays, user context data are scattered around due to the significant growing in smart phones, mobile apps, sensing technology and social networking, how to combine these data to leverage information about users are important and promptly needed, which motivated the topic of my thesis: Applying Personal Cloud in Existing Mobile Context Aware Framework.


Role: Developer & Researcher

Teams: NA

Duration: 3 months


Overview:


In my thesis project I reviewed the approaches in existing context aware platforms, and reformed the context aware framework by absorbing previous fundamental technical aspects into personal cloud. By using a user-centric context collector approach, I enhanced the overall applications intelligence and reduced the privacy and security problem.


A prototype application has also been developed for the purpose of evaluating the Context widget and obtaining hands-on experience of building a context-aware application. The function of the mobile application is fetching the content from a web site for film on TV today. And based on historical selected liked film database, contexts on user film preferences are auto-generated and context aware adaption is allowed for updating the film list order based on its reasoning logic.
