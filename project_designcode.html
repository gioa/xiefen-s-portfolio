MHCID Website Redesign

Project Background:
The assignment is to redesign the MHCID homepage. The challenge is to change the style with css while not updating the html file. It is like css zen garden excercise.


Role: Designer and Developer

Teams: NA

Duration: 1 week


Overview:

To give the MHCID homepage a facelift, I first followed the current design trends, which includes large image, responsive design, and longer scrolling site with clear sections. So my 1st design starts with the logo and navigation bar on the top, followed by a large image (could be the background image for the logo and menu); then the notification section, the program description section with images to serve the content, and the news section; lastly, the footer has the general elements like logo, contact information, subscribe functions, and sponsor information. It also includes the navigation bar in case user wants to jump to other page after scrolling down.

I want to try different pattern for the second design. I really like google's material design. So I tried the flat design here. I hide the navigation in the left top corner, then I have a major area with different color to highlight the pages I want the user to click into. These are also the elements from the menu bar, with a bigger appearance to motivate users to click into. My inspiration comes from Google Design .

Next I move to css styling. I decided to do the first design since the elements did not change much. I also layout the visual design with photoshop:






After the design I started to code, as I said, the exercise was very challenging as we could not change the html file. So it does not look exactly like the visual design image, but I did my best in the time limit.
